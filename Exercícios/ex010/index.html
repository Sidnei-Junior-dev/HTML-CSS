Trabalhando com links

Links Externos

usando a tag -a href="url do link"- conseguimos direcionar você para outro site Porém é recomendado usar com os parâmetros -target="_blank"- e -rel="external"- para abrir o site em questão em outra aba de navegação.

Por exemplo com este link aqui sem os parâmetros. e este link com os parâmetros.

Links Internos

Podemos acessar os links internos também, isso sem digitar a url do site.

para isso bastar linkar com a tag -a href="link em questão"- + -rel="next" para indicar q a próxima página.-

Links para Downloads

Para colocar um link para download devemos ter o arquivo em questão e linka-lo com -a href=""-

Por exemplo este link em Pdf.

E este link em Zip.

Certos arquivos não são instalados e seram abertos no próprio navegador, portanto não necessita do parametro -download="arquivo em quetão"-.

também temos que declarar o tipo do arquivo que estamos fazendo download portanto -type="application/tipo do arquivo"- para achar o tipo do arquivo basta acessar o site:http://iana.org/assigments/media-type
